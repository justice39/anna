# Fonts

Download these Google Fonts and place the TTF files here:

## Required

- `InstrumentSerif-Regular.ttf` — https://fonts.google.com/specimen/Instrument+Serif
- `InstrumentSerif-Italic.ttf` — same family, italic style

## Note

The `google_fonts` package will fetch these at runtime if you skip this step, but
bundling the TTFs makes first launch faster and lets the app work offline.

After adding the files, run `flutter pub get` again.
